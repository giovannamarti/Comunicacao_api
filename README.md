# Comunicacao_api

Post agendar comunicação
Get buscar Status comunicação
Patch alterar Status comunicação

## Endpoints da Aplicação
-
**GET /api/comunicacao/: Buscar Status Comunicação.
-
**POST /api/comunicacao/agendar: Agendar Comunicação

**PATCH /api/comunicacao/cancelar: Alterar Status Comunicação
